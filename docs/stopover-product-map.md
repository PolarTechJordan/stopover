# 中转游 Stopover 产品脑图

## 1. 产品定位（一句话）

> **以休息室为信任锚 + 行李全托管 + 模块化城市服务**，把中转 6–48h 的"机场等待"升级为"轻量级目的地体验"。

---

## 2. 核心痛点与产品对策

| 痛点 | 产品对策 |
|------|---------|
| 时间焦虑（6–48h 怎么过） | 标准化时段套餐：3h/4h/6h/8h/12h 自由组合 |
| 行李焦虑（拖箱出不去） | 行礼包全托管，从柜台到贵宾厅/酒店到登机口 |
| 决策焦虑（怕踩坑、怕误机） | 固定路线 + 误时保护 + 误机保障金 |
| 体验焦虑（陌生中转地） | 中文/英文向导 + 经典路线 + 当地商户权益 |

---

## 3. 用户分群 × 套餐匹配

```
                    ┌─ 探索型商旅（8–24h）─── 微游包 + 包车
                    │
中转 6–48h 旅客 ────┼─ 家庭中转（12–36h）──── 过夜包 + 钟点房
                    │
                    └─ 休息补能型（6–12h）─── 轻享包 / 睡眠舱
```

---

## 4. 三档核心套餐

| 套餐 | 适合时长 | 核心组成 | 价格带 |
|------|---------|---------|--------|
| 🪑 **轻享包** | 6–8h | 休息室 3h + 行李寄存 + 快速安检 | ¥260–320 |
| 🚌 **微游包** | 10–18h | 休息室 2h + **行李托管** + 城市 4h 游 + 接送 | ¥680–880 |
| 🛏️ **过夜包** | 12–36h | 休息室 1h + **行李托管** + 酒店/钟点房 + 接送 | ¥780–1200 |

## 7. 与竞品差异化

| 维度 | 土航中转游 | 携程 × 首都机场 | **本产品** |
|------|----------|---------------|-----------|
| 航司限制 | ❌ 仅限土航联程 | ❌ 仅北京 | ✅ 全中转开放 |
| 机场内 | 限制多 | 部分 | ✅ 全机场生态 |
| 行李托管 | 无 | 无 | ✅ **全托管** |
| 误机保障 | 弱 | 弱 | ✅ 赔付 + 改签 |
| 套餐化 | 单点 | 单点 | ✅ 3 档 + 增值 |
| 城市选择 | 限伊斯坦布尔 | 限北京 | ✅ 多枢纽 |

## 8. 商业模型（粗算）

- **目标客单**：¥450+（当前 ¥200）
- **首年枢纽**：3–5 个
- **首年目标订单量**：5–10 万单（PoC 枢纽合计）
- **毛利结构**：套餐毛利 30–35%，增值项毛利 25–40%
